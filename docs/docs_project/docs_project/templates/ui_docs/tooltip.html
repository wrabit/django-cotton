API

ui.tooltip

Properties

Name Type Values Default Description
position String top, bottom, left, right top The position of the tooltip relative to the trigger element.
:offset Integer Any integer 4 The gap between the trigger and the tooltip in pixels.
attrs Dict HTML attribute dictionary {} Passes HTML attributes directly to the wrapper <div> element.

Slots

Name Required Description
slot (default) Yes The element that triggers the tooltip (e.g., a button, icon, or text).
content Yes The content (text or HTML) to display inside the tooltip.
