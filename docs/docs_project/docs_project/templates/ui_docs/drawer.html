Drawer A panel that slides in from any edge of the screen for secondary content, forms or details. Traps focus and locks body scroll while open; requires the Alpine.js focus plugin. Basic Usage Sides Sizing Reference

The default slot holds the trigger; open it from any element inside with @click="drawerOpen = true". Put the body in the content slot and any actions in the footer slot.
