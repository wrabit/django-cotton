Height

Control the initial height using the height attribute. This sets a fixed height, which will be overridden if :autoresize="True" is used and the content grows.

{% cotton_verbatim %}{% verbatim %} {% endverbatim %}{% endcotton_verbatim %}

Autoresize

Use the :autoresize="True" attribute to make the textarea automatically adjust its height based on the content. Requires Alpine.js.

{% cotton_verbatim %}{% verbatim %} {% endverbatim %}{% endcotton_verbatim %}

Non-resizable

Prevent the user from manually resizing the textarea using :resizable="False".

API

ui.textarea

Properties

Name Type Values Default Description
:autoresize Boolean True, False False Automatically adjusts height to fit content. Requires Alpine.js.
size String xs, sm, base, lg, xl base Sets the font size and padding.
height String xs, sm, base, lg, xl base Sets the initial fixed height. Overridden by autoresize.
:resizable Boolean True, False True Allows or prevents manual resizing by the user via CSS.
attrs Dict HTML attribute dictionary {} Passes HTML attributes (like placeholder, name, id, rows, etc.) directly to the underlying <textarea> element via {{ attrs }}.

Slots

Name Default Description
slot (default) '' The initial content/value of the textarea.
