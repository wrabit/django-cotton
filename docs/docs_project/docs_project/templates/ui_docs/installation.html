Installation Set up Cotton UI in your Django project with Tailwind CSS and Alpine.js. Requirements Install Packages Django Settings Base Template Tailwind Setup Accent Colors

Before installing Cotton UI, ensure your project has:

  • Python 3.8+
  • Django 4.2+
  • Tailwind CSS v4 configured in your project
  • Alpine.js v3 for interactive components

Install Cotton UI and its dependencies via pip:

{% cotton:verbatim %}{% verbatim %} pip install django-cotton-ui {% endverbatim %}{% endcotton:verbatim %}

This will also install django-cotton as a dependency.

Add both django_cotton and cotton_ui to your INSTALLED_APPS:

{% cotton:verbatim %}{% verbatim %} INSTALLED_APPS = [ # ... "django_cotton", "cotton_ui", # ... ] {% endverbatim %}{% endcotton:verbatim %}

The order matters: django_cotton should come before cotton_ui.

Your base template needs to include the Cotton UI stylesheet, JavaScript, and Alpine.js. Here's a minimal setup:

{% cotton:verbatim %}{% verbatim %} {% load static %} {{ slot }} {% endverbatim %}{% endcotton:verbatim %}

Required Alpine.js Plugins

Cotton UI components use these Alpine.js plugins:

  • @alpinejs/collapse - For accordion and collapsible content
  • @alpinejs/focus - For focus trapping in dialogs and dropdowns
  • @alpinejs/mask - For input masking (datepicker, etc.)

Plugin scripts must be loaded before the Alpine.js core script.

Configure Tailwind to scan Cotton UI component templates. In your main CSS file:

{% cotton:verbatim %}{% verbatim %} @import "tailwindcss"; /* Scan your templates */ @source "./your_app/templates/**/*.html"; /* Scan Cotton UI component templates */ @source "./path/to/cotton_ui/templates/**/*.html"; {% endverbatim %}{% endcotton:verbatim %}

To find the Cotton UI templates path, run:

{% cotton:verbatim %}{% verbatim %} python -c "import cotton_ui; print(cotton_ui.__path__[0] + '/templates/**/*.html')" {% endverbatim %}{% endcotton:verbatim %}

Cotton UI uses a customizable accent color system. Define your accent in app.css using Tailwind v4's @theme directive:

{% cotton:verbatim %}{% verbatim %} @import "tailwindcss"; @theme { /* Accent colors (teal by default) */ --color-accent: var(--color-teal-500); --color-accent-content: var(--color-teal-600); --color-accent-foreground: var(--color-white); } /* Dark mode overrides */ @layer theme { .dark { --color-accent: var(--color-teal-400); --color-accent-content: var(--color-teal-300); --color-accent-foreground: var(--color-teal-950); } } {% endverbatim %}{% endcotton:verbatim %}

For more customization options, including different accent colors and gray palettes, see the Theming guide.
