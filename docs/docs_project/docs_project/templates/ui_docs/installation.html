Installation Set up Cotton UI in your Django project with Tailwind CSS and Alpine.js. Requirements Install Packages Django Settings Base Template Tailwind Setup Accent Colors

Before installing Cotton UI, ensure your project has:

  • Python 3.8+
  • Django 4.2+
  • Tailwind CSS v4 with a build step. The same setup any Tailwind project uses, but the Play CDN and v3 are not supported: the kit's classes are compiled from your @source-scanned templates (set up below).
  • Alpine.js v3 for interactive components

Install Cotton UI and its dependencies via pip:

{% cotton:verbatim %}{% verbatim %} pip install django-cotton-ui {% endverbatim %}{% endcotton:verbatim %}

This will also install django-cotton as a dependency.

Add both django_cotton and django_cotton_ui to your INSTALLED_APPS:

{% cotton:verbatim %}{% verbatim %} INSTALLED_APPS = [ # ... "django_cotton", "django_cotton_ui", # ... ] {% endverbatim %}{% endcotton:verbatim %}

The order matters: django_cotton should come before django_cotton_ui.

Your base template needs to include the Cotton UI stylesheet, JavaScript, and Alpine.js. Here's a minimal setup:

{% cotton:verbatim %}{% verbatim %} {% load static %} {{ slot }} {% endverbatim %}{% endcotton:verbatim %}

Required Alpine.js Plugins

Cotton UI components use these Alpine.js plugins:

  • @alpinejs/collapse - Accordions, the collapse component and the mobile navbar menu
  • @alpinejs/focus - Focus trapping in dialogs and drawers, and keyboard focus in dropdowns and menus

Plugin scripts must be loaded before the Alpine.js core script.

Configure Tailwind to scan Cotton UI component templates. In your main CSS file:

{% cotton:verbatim %}{% verbatim %} @import "tailwindcss"; /* Cotton UI: accent + component tokens + x-cloak. Override the accent below to brand it. */ @import "./path/to/django_cotton_ui/static/cotton-ui/cotton-ui.css"; /* Scan your templates */ @source "./your_app/templates/**/*.html"; /* Scan Cotton UI component templates */ @source "./path/to/django_cotton_ui/templates/**/*.html"; {% endverbatim %}{% endcotton:verbatim %}

To find the Cotton UI templates path, run:

{% cotton:verbatim %}{% verbatim %} python -c "import django_cotton_ui; print(django_cotton_ui.__path__[0] + '/templates/**/*.html')" {% endverbatim %}{% endcotton:verbatim %}

A neutral zinc accent ships by default via the cotton-ui.css import above. To brand the kit, override --color-accent in your own @theme after that import (with a .dark value for dark mode):

{% cotton:verbatim %}{% verbatim %} @theme { --color-accent: var(--color-teal-500); --color-accent-content: var(--color-teal-600); --color-accent-foreground: var(--color-white); } @layer theme { .dark { --color-accent: var(--color-teal-400); --color-accent-content: var(--color-teal-300); --color-accent-foreground: var(--color-teal-950); } } {% endverbatim %}{% endcotton:verbatim %}

For more customization options, including different accent colors and gray palettes, see the Theming guide.
